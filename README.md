Para que a API funcione corretamente:
  1.Faça um NPM init -y na pasta em que a API será mantida
  2.Instale as dependencies que deixei a mostra no package.json
  3.Importe o arquivo do banco no seu SGBD (Código sujeito a mudanças dependendo do banco utilizado)
  4.Troque os dados de segurança do arquivo .env para os seus dados locais
